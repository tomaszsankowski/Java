# Lab 1: Java Collections

This mini-project demonstrates the use of the Java Collections Framework, specifically focusing on `Set` implementations.

## Features

- **HashSet:** Unordered collection of unique elements.
- **TreeSet:** Ordered collection of unique elements.
- **Custom Sorting:** Demonstrates default sorting and alternative sorting mechanisms using `Comparator` and `Comparable`.

## How to Run

1. Open the project in your IDE.
2. Run the `Main` class located in `src/main/java/org/example/Main.java`.
3. Provide a command-line argument (`1`, `2`, or `3`) to select the sorting mode:
   - `1`: No sorting (`HashSet`)
   - `2`: Default sorting (`TreeSet`)
   - `3`: Alternative sorting (`TreeSet` with custom comparator)
