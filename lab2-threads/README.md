# Lab 2: Threads

This mini-project introduces multi-threading in Java, focusing on concurrent execution and thread management.

## Features

- **Thread Creation:** Demonstrates how to create and start threads in Java.
- **Concurrency:** Shows concurrent execution of tasks, such as calculating prime numbers or performing mathematical operations using a `Calculator` class.
- **Synchronization:** Basic concepts of thread safety and synchronization.

## How to Run

1. Open the project in your IDE.
2. Run the `Main` class located in `src/main/java/org/example/Main.java`.
3. Observe the console output to see the concurrent execution of threads.

## Technologies

- Java
